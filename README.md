# SkyCast - Weather Web Application

SkyCast is a **full stack weather web application** built using **React (Frontend)** and **Django (Backend)**.
The application allows users to search for cities and view real-time weather information.

This project demonstrates how a **React frontend communicates with a Django backend API**.

---

# 🚀 Project Features

* Search weather by city
* Display temperature
* Show weather conditions
* Responsive UI
* React + Django API integration

# ⚙️ Installation

## 1️⃣ Clone Repository

```
git clone https://github.com/dineshwaran2004/MEMBER-Django-Project.git
```

```
cd MEMBER-Django-Project
```

---

# 🖥 Backend Setup (Django)

```
cd backend
pip install django djangorestframework
python manage.py migrate
python manage.py runserver
```

Backend will run on:

```
http://127.0.0.1:8000
```

---

# 🌐 Frontend Setup (React)

Open another terminal.

```
cd Frontend/react-frontend
npm install
npm start
```

Frontend will run on:

```
http://localhost:3000
```
